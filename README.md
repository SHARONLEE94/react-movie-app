## Styled Component
👉 Styled Components는 CSS-in-JS 라이브러리 중 하나로, JavaScript 코드 안에서 직접 스타일을 작성할 수 있도록 도와주는 도구. React에서 많이 사용되며, 컴포넌트 단위로 스타일을 적용할 수 있어 유지보수가 편리하다
`npm install --save styled-components`
